# Examples Directory

This directory contains proof artifacts for CV and portfolio purposes.

## Contents

### screenshot_1.png
Screenshot of the KPI snapshot CSV showing key metrics (MRR, ARR, LTV:CAC, etc.)

**To generate**: 
1. Run `python run.py`
2. Open `outputs/kpi_snapshot.csv` in Excel
3. Take a screenshot showing the metrics

### sample_report.pdf
Auto-generated summary report exported to PDF format.

**To generate**:
1. Run `python run.py`
2. Open `outputs/summary_report.txt`
3. Export/print to PDF
