# Sample Visuals

This directory contains sample visualizations and screenshots of the SaaS Analytics Engine outputs.

## Recommended Visuals to Add

1. **funnel_visualization.png** - Funnel chart showing Sign-up → Activated → Paid → Retained
2. **cohort_heatmap.png** - Heatmap of cohort retention matrix
3. **mrr_trend.png** - Line chart of MRR over time
4. **mrr_bridge_waterfall.png** - Waterfall chart of MRR components
5. **ltv_cac_by_channel.png** - Bar chart of LTV:CAC ratios by acquisition channel
6. **scenarios_comparison.png** - Line chart comparing 6 scenarios

## How to Generate

Use tools like:
- **Excel/Google Sheets** - Import CSVs and create charts
- **Tableau** - Connect to CSV files
- **Python (matplotlib/seaborn)** - Generate programmatically
- **Power BI** - Create interactive dashboards

## For Recruiters

Screenshots in this folder provide instant visual proof of the analytics capabilities without requiring code execution.
